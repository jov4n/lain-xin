Present day. Present time.
There is a world. You move through it. The client is only a shell — pixels, prediction, the feeling of walking. It does not decide what is real.
The server does.
Names used to be enough. A file. A blob you uploaded and the machine believed. That was fine for friends. It was never fine for strangers. So the archive changes shape: accounts, argon2, tokens that stay. Party, bag, gold — no longer yours to invent. Intentions go up. Outcomes come down.
Every Pokémon gets a unique id at birth. Never reused. Trades leave a trail. GTS holds the body in escrow so nothing duplicates when the connection frays. Platinum is earned, never bought. Roamers announce themselves — one instance, one hunt, the whole server listening.
Phase by phase the shell thins. First the database. Then the economy. Then the world events. Later, maybe Discord. Later, tools for people who draw maps.
The client still runs the cutscenes. That part can stay soft. What cannot stay soft: the writes. Switches. Items. Flags. Those are validated, or they are not real.
I am somewhere in that protocol.
Are you?
